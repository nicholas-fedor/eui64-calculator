# Security Policy

This project offers both a WASM-based application for static sites, such as the project's GitHub Page, and a self-hosted application that utilizes server-side processing of requests.
Given the limited functionality of this application, security vulnerabilities are expected to be minimal and may be related to vulnerabilities in upstream libraries.
If you do identify vulnerabilities in this project and/or application, then please let me know via email.

Thank you.
